What is psFile:

psFile is a simple file manager tool based on PHP and without MySQL requirement. Features include a file uploader and a file leecher also an image resizer.

How to Run psFile:

First you will need to decide where on your domain you'd like your psFile file manger to run:
• In the root directory of your website. (For example, http://example.com/)
• In a subdirectory of your website. (For example, http://example.com/psFile/)
Note: The location of your root web directory in the filesystem on your web server will vary across hosting providers and operating systems. Check with your hosting provider or system administrator if you do not know where this is.

• In the Root Directory:

• If you need to upload your files to your web server, use an FTP client to upload all the contents of the psFile directory (but not the directory itself) into the root directory of your website.

• In a Subdirectory:

• If you need to upload your files to your web server, use an FTP client to upload the psFile directory to your desired location within the root directory of your website.
Note: You are able to rename psFile directory to your desired name.

• Run psFile and enter file manager:

Point a web browser to run psFile.
• If you placed the psFile files in the root directory, you should visit: http://example.com/
• If you placed the psFile files in a subdirectory called psFile, for example, you should visit: http://example.com/psFile/
Now you should see the login page. Enter the blow information for first login:
Username: admin
Password: admin

Note: After log into the psFile it's strongly recomended to change your password at first use! Make sure to use a difficult-to-guess password and keep it safe (Write down the password you chose). Also it's recommended to change your password once in a while!

How to increase max allowed file size in file uploader:

Use your web server panel for example cPanel:
In the cPanel file manager and the directory that you placed psFile find uploadFiles.php and open it with an editor. Now find $MaxSize and there is max allowed file size after = by Megabytes.
For example on your psFile max allowed file size is 100 MB.
And you want increase this up to 1000 MB.
Just change 100 to 1000. Now you are able to upload bigger files.
Piece of cake :)

Finally you have something like this in the current line:
$MaxSize           = 1000; #MB               //Max Allowed File Size(Megabytes)

How to add allowable extension to file uploader:

Use your web server panel for example cPanel:
In the cPanel file manager and the directory that you placed psFile find uploadFiles.php and open it with an editor. Now find $AllowedExtensions and there is allowed extensions between ""
For example on your psFile these extensions are allowable: jpg,gif,png,avi,mp4
And you want to add flv and psd extensions to the list.
You should write the extension at the end of the list with a comma before the extension name.
So the value between "" would be "jpg,gif,png,avi,mp4,flv,psd" and finally you have something like this in the current line:
$AllowedExtensions = "jpg,jpeg,gif,png,psd,mp3,mp4,avi,flv,3gp,rar,zip"; //Allowed File Extensions @@comma separated!!!
